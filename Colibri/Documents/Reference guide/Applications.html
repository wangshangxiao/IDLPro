Applications

This section describes the ENVI/IDL applications developed under the Open Source Initiative.

Alphabetic list of the E/I-OSI general applications

Routine NameDescription
EOSap_ClassifyThis application procedure performs image classification on a list of images by using a set of 'Regions Of Interest' (ROIs) as ground reference data.
EOSap_Create_Synthetic_Image
This application procedure creates a synthetic hyperspectral image from spectra selected from an ENVI spectral library.
EOSap_DSM2DTMThis application procedure extracts the DTM raster from the specified DSM raster image.
EOSap_Interpolate_EmptyThis application procedure fills the empty pixels in the input image(s) by interpolating.
EOSap_SSIREE This application procedure extracts the Spectrally Purest Pixels from a set of images using the Spatial/Spectral Iterative Rule base Endmember Extraction algorithm.

Alphabetic list of the E/I-OSI toolbox applications

Routine NameDescription
EOSap_ClassEditorUse this application to edit and modify ENVI classified images.
EOSap_SmoothingThis application will perform spectral smoothing of hyperspectral images.
