IOScc_aSA

Syntax | Return Value | Arguments | Keywords | Version History

This function will calculate the maximum SA value between a reference spectrum and its surrounding spectra as defined by the kernel size.

Syntax

Value = IOScc_aSA(pBank, iActual, Radius)

Return Value

The calculated maximum SA value.

Arguments

pBank

An n element pointer array holding the spectral slice for which to calculate the SA values. Here n equals to the diameter of the convolution kernel. Each pointer points to a 2 dimensional array s x b, with s the number of samples and b the number of bands.

iActual

An index into the spectral slice indicating the central pixel for which the maximum SA value has to be calculated.

Radius

A value holding the radius of the convolution kernel. The diameter is calculated as Radius*2 + 1.

Keywords

None.
