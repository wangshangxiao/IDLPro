EOSocImages::Extract_Libraries_From_Rois

Syntax | Arguments | Keywords | Examples | Version History

This procedure method extracts the spectral information from the specified images by means of the assigned ROIs.

Syntax

Obj->[EOSocImages::]->Extract_Libraries_From_Roi, ROI_DIR=string [, aROICOLORS=variable] [, aROINAMES=variable] [, BANDSIX=array] [, pLIBRARIES=variable] [, SELECTIX=array]

Arguments

ROI_DIR

Set this keyword to a string holding the directory where the ROI files are located.

Keywords

aROICOLORS

A vector or array of vectors holding the RGB color values for the different ROIs.

aROINAMES

A string or array of strings holding the names of the different ROIs.

BANDSIX

Set this keyword to a vector holding the band indices for which the spectral values have to be extracted. If not specified all bands are used.

pLIBRARIES

Set this keyword to a variable in which a pointer array is returned. Each pointer points to a spectral library structure array with layout:
  • ImageFile: a string holding the full path and name of the image file from which the spectrum was extracted.
  • RoiName: a string holding the ROI name of the extracted spectrum.
  • Spectra: a array holding the extracted spectrum.

SelectIx

The image index or indices in the image list, which was specified during initialization of the object, for which data has to be retrieved. If not specified the information of the first image is returned.

Examples

An extensive example on how to use this application procedure can be found in the examples directory: ../Colibri/Examples/IOSex_ROI_Extract_Libraries
