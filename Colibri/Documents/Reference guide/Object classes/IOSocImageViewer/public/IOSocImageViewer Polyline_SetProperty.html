IOSocImageViewer::Polyline_SetProperty

Syntax | Arguments | Keywords | Version History

This procedure method is used to add, clear or close a polyline to the view.

Syntax

Obj->[IOScocImageViewer::]Polygons_SetProperty [, ADDVERTICES=array] [, /CLEAR][, /CLOSE], [COLOR=vector]

Arguments

None.

Keywords

ADDVERTICES

Set this keyword to a 2-by-n array holding the sample and line locations of the polyline to be added to the view.

CLEAR

Set this keyword to remove all existing polylines from the view.

CLOSE

Set this keyword to close the existing polyline in order to obtain a polygon.

COLOR

Set this keyword to a 3 element vector [red, green, blue] specifying the color used to draw polylines.
