IOSocImageViewer::Inform_User

Syntax | Arguments | Keywords | Dependencies | Version History

This private procedure method is used to inform the user of the class 'IOScocImageViewer' about an event that occured in one of the created windows.

Syntax

Obj->[IOSocImageViewer::]Inform_User, event
This procedure is called when an image event occurs.

Arguments

event

This variable holds the event structure generated by the Draw widget:
{WIDGET_DRAW, ID:0L, TOP:0L, HANDLER:0L, TYPE: 0, X:0L, Y:0L, PRESS:0B, RELEASE:0B, CLICKS:0L, MODIFIERS:0L, CH:0B, KEY:0L }
For more info: IDL help - WIDGET_DRAW reference.

Keywords

None.

Dependencies

The user of the 'IOSocImageViewer' class should have a procedure method defined called 'ImageViewer_User_Info' which accepts the 'ImageViewerInfo' argument. The 'ImageViewerInfo' argument has following structure:
  • Central_Pixel_Location: A two element array holding sample and line location of the central pixel.
  • Actual_Pixel_Location: A two element array holding sample and line location of the actual pixel.
  • Event: Widget Draw event structure
