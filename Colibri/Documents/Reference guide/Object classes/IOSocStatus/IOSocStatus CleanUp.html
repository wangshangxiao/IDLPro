IOSocStatus::CleanUp

Syntax | Arguments | Keywords | Version History

The IOSocStatus::Cleanup procedure method performs all cleanup on the object.  

Syntax

OBJ_DESTROY, Obj

or

Obj->[IOSocStatus::]Cleanup     (In a lifecycle method only.)

Arguments

None.

Keywords

None.
