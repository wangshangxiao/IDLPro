IOSocStatus::Progress_Clear

Syntax |  Arguments | Keywords | Version History

The IOSocStatus::Progress_Clear procedure method is used to reset the progress bar. This means the progress bar is cleared, the status lamp is set to green and the info on the status line is cleared.

Syntax

Obj->[IOSocStatus::]Progress_Clear [, /FAST] [, /RESET] [, /SLOW]

Arguments

None.

Keywords

FAST

Set this keyword to clear the ‘fast’ progress bar.

RESET

Set this keyword to clear both the ‘fast’ and ‘slow’ progress bar, to clear the status line and to set the LED status to green.

SLOW

Set this keyword to clear the ‘slow’ progress bar.
