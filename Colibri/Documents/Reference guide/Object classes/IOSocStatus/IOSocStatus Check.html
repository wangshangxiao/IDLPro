IOSocStatus::Check

Syntax | Return Value | Arguments | Keywords | Version History

The IOSocStatus::Check function method allows the requestor to check the status of the previous reported state. This allows the requestor to decide on the next processing step. 

Syntax

Result=Obj->[IOSocStatus::]Check [, (‘Ok’ | 'Error' | 'Warning' | 'Busy' | 'Request' | 'DontKnow')]

Return Value

The function method returns 1 if the passed state matches the state of the last reported status otherwise 0 is returned. If no or an unknown string value is specified 0 is returned.

Arguments

A string holding the state which the function method should check for. Possible string values are: ‘Ok’, 'Error', 'Warning', 'Busy', 'Request', 'DontKnow'.

Keywords

None.
