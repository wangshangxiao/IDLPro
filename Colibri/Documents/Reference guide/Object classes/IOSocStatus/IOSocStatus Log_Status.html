IOSocStatus::Log_Status

Syntax | Arguments | Keywords | Version History

The IOSocStatus::Log_Status procedure method updates the status window according the state and info passed to the procedure. If requested the status information iswritten to the log file. . 

Syntax

Obj->[IOSocStatus::]Log_Status [, State] [, Info] [, Module] [, DIRECTORY=string] [, FILE_PREFIX=string] [, /LOG]

Arguments

State

A string holding the actual state to which the status window has to be updated. The LED colour and the prefix of the reported info is set according to the reported state. Following states can be reported:

StateLED colorPrefix
OkGreen
ErrorRed*** ERROR:
WarningOrange** WARNING:
BusyYellow* INFO:
RequestBlueREQUEST:
DontKnowDark???:

NoteIf no arguments are specified the default 'DontKnow' is reported.

Info

A string holding additional information about the new state. This information is reported on the status line.

Module

A string holding the name of the calling procedure or function.

Keywords

DIRECTORY

A text string holding the path where the log file has to be written. If specified the log file is created and opened. 

FILE_PREFIX

A text string holding the file prefix of the log file to be created. If not specified a default prefix is given to the log filename. The log filename is automatically generated and contains the prefix, date and time, and the extension '.txt'.

LOG

Set this keyword to write the actual status information to the log file.
