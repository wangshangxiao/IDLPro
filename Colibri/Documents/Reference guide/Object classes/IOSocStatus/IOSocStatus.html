IOSocStatus

Superclasses | Creation | Methods | Version History

This class is used to create and manage a status information window. This window contains a status LED, status line, progress bar and logo button. The user application has the possibility to update the status window according the actual status of the running application by changing the LED color, placing text on the status line and by steering the progress bar. When the logo button is pressed a designer info window will appear that contains useful information of the designer who wrote the application. Once the running process has finished, the status window can be closed by pressing the status LED.
