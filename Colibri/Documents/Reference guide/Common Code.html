Common Code

This section describes the ENVI/IDL common code developed under the Open Source Initiative.

Alphabetic list of the E/I-OSI common code.

ENVI based routines
Routine NameDescription
EOScc_ClassDoitThis procedure performs image classification by using the ENVI procedure "Class_Doit".
EOScc_Find_RGB_BandsThis procedure will search the Red, Green and Blue band indices for the specified image.
EOScc_SVMThis procedure performs Support Vector Machine classification by using the ENVI procedure "Envi_SVM_Doit".


IDL based routines
Routine NameDescription
IOScc_aRMSEThis function will calculate the maximum RMSE value between a reference spectrum and its surrounding spectra as defined by the kernel size.
IOScc_aSAThis function will calculate the maximum Spectral Angle (SA) value between a reference spectrum and its surrounding spectra as defined by the kernel size.
IOScc_Handle_Input_DirThis function is used to locate the image files to be handled by an application.
IOScc_Handle_Output_DirThis function is used to locate the output directory to be used by an application.
IOScc_InitThis function will check if a status object representing the status window exist, if not it will create one.
